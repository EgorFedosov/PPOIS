# Документация системы

## Предметная область
Система моделирует приготовление яичницы с сущностями:
- `Stove`
- `Pan`
- `Spatula`
- `Egg`
- `Oil`
- `Spices`

## Операции
В `CookingProcess` реализованы операции варианта:
- `heat_pan()` - разогрев сковороды;
- `break_eggs()` - разбивание яиц;
- `add_spices()` - добавление приправ;
- `fry_eggs()` - обжаривание яиц;
- `mix_and_serve()` - перемешивание и подача.

Дополнительно есть `add_oil()` для использования сущности масла.

## Конечный автомат
Состояния процесса:
1. `IDLE`
2. `HEATING`
3. `COOKING`
4. `SEASONING`
5. `FINISHING`
6. `SERVED`

Некорректные переходы обрабатываются исключениями из `src/exceptions.py`.

## Сохранение состояния
Снимок состояния формируется методом `snapshot()` и сохраняется в `data/state.json` через `JsonStorage`.

При запуске выполняется восстановление через `CookingProcess.from_snapshot(...)`.

## CLI
`src/ui/menu.py` предоставляет консольные команды для всех операций и выполняет автосохранение после успешного действия.
